Variables

Variables are containers for storing data...

You can declare them with let, const, or var.

  • let: Block-scoped
  • const: Immutable after initialization
  • var: Function-scoped
let name = "Kelyzor";
Declaring Variables

To declare a variable, use let, const, or var...

Examples of declarations:

  • let x = 5;
  • const y = 10;
  • var z = "test";
const PI = 3.14;
